# Privacy Policy

**Effective Date:** January 2025
**Last Updated:** January 2025
**Product:** Delta Saraswati
**URL:** https://delta-saraswati.vercel.app

---

## 1. Introduction

Delta Saraswati is a proprietary dark-mode cryptocurrency perpetuals trading dashboard that provides real-time market data, AI-powered analysis, technical indicators, and aggregated news for informational purposes. This Privacy Policy explains how the application handles data.

The key principle is straightforward: **Delta Saraswati does not collect, store, or process any personal data.** There is no user authentication, no account creation, no login system, and no mechanism for collecting personally identifiable information.

---

## 2. Information We Do Not Collect

Delta Saraswati does **not** collect or process any of the following:

- Names, email addresses, phone numbers, or any personally identifiable information
- Login credentials or account information (no authentication system exists)
- Payment or financial account details
- IP addresses for tracking or profiling purposes
- Browsing history or behavioral analytics
- Location data
- Device fingerprints for user identification

---

## 3. How the Application Works

Delta Saraswati operates as a read-only dashboard that aggregates publicly available data from third-party sources. All data flows from external APIs to the user interface. No user-generated data flows in the reverse direction.

The application fetches and displays:

- **Market data** (prices, order books, funding rates, open interest) from public exchange APIs
- **AI-generated market analysis** produced by large language model inference
- **News articles** aggregated from public RSS feeds
- **Market sentiment indicators** from public index APIs
- **Technical indicators** computed from public price data

---

## 4. Third-Party Services

Delta Saraswati integrates with the following third-party services to provide its functionality. Each service has its own privacy policy, and users are encouraged to review them independently.

### 4.1 Delta Exchange India
- **Purpose:** Source of live market data for cryptocurrency perpetual contracts, including prices, order books, funding rates, and open interest.
- **Data flow:** The application makes read-only API calls to retrieve public market data. No user information is transmitted to Delta Exchange through this application.
- **Note:** "Trade Now" buttons within the dashboard redirect users to the Delta Exchange platform. Any interaction on Delta Exchange is governed by their own terms and privacy policy.

### 4.2 Groq (Meta Llama 3.3 70B)
- **Purpose:** AI inference for generating market analysis, trade signals, and sentiment summaries.
- **Data flow:** The application sends market data context (prices, indicators, news headlines) to Groq's inference API. No personal or user-specific data is included in these requests.
- **Privacy policy:** https://groq.com/privacy-policy

### 4.3 Vercel
- **Purpose:** Application hosting and edge caching via Vercel KV.
- **Data flow:** Vercel serves the application and provides short-lived key-value caching for API responses. Standard web server logs may be generated by Vercel's infrastructure as part of normal hosting operations.
- **Privacy policy:** https://vercel.com/legal/privacy-policy

### 4.4 RSS News Feeds
- **Sources:** CoinDesk, CoinTelegraph, Decrypt
- **Purpose:** Aggregation of cryptocurrency news headlines and summaries.
- **Data flow:** The application fetches publicly available RSS feeds. No user data is transmitted to these news sources.

### 4.5 Fear and Greed Index API
- **Purpose:** Market sentiment indicator.
- **Data flow:** Read-only retrieval of the current index value. No user data is transmitted.

## 5. Cookies and Local Storage

Delta Saraswati does **not** use tracking cookies, advertising cookies, or analytics cookies.

The application may use minimal local storage or service worker caching solely for Progressive Web App (PWA) functionality, such as caching static assets for offline performance. This data remains entirely on the user's device and is never transmitted to any server.

---

## 6. Data Retention

- **API response caching:** Market data and AI analysis responses are cached briefly (approximately 30 seconds to 1 hour, depending on the data type) in Vercel KV to reduce redundant API calls and improve performance. Cached data is automatically purged after expiration.
- **Persistent user data:** None. No user data is stored persistently in any database, cache, or file system.

---

## 7. Data Security

Although Delta Saraswati does not handle personal data, the application is served exclusively over HTTPS to ensure encrypted communication between the user's browser and the hosting infrastructure. API calls to third-party services are also made over encrypted connections.

---

## 8. Children's Privacy

Delta Saraswati does not knowingly target or collect information from individuals under the age of 18. The application has no mechanism for collecting information from any user, regardless of age.

## 9. International Users

Delta Saraswati is accessible globally but is designed primarily for the Indian market. The application does not collect personal data from users in any jurisdiction. Third-party services may operate infrastructure in various countries, subject to their own privacy policies.
